Inline a function/method — the reverse of extract. This is agent-driven (no Gradle task). Steps: 1) Run `./gradlew opsx-usages -Psymbol=Name` to find all call sites. 2) Run `./gradlew opsx-find -Psymbol=Name` to get the implementation. 3) Replace each call site with the function body, adjusting parameters. 4) Delete the original function. 5) Run `./gradlew opsx-usages -Psymbol=Name` again to verify no remaining references.
